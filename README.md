# MINVIM

## Minimal nvim distro

### Dependencies

Install `fzf` `curl` `nvim` and `rg`.
`brew install fzf curl nvim rg`
or
`apt-get install fzf curl nvim rg`
or the equivalent.

### Install
* Create & open config file `md -p ~/.config/nvim && nvim ~/.config/nvim/init.lua`
* Copy content of `init.lua` file from this repo.
* Press `P` `Z` `Z` on nvim.
