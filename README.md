# MINVIM

### Dependencies

##### Ensure you have `fzf` `curl` `nvim` and `rg`.
* Mac 
`brew install fzf curl nvim rg`

* Linux 
`apt-get install fzf curl nvim rg`

### Install
* Run this
  ```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/jelohe/minvim/refs/heads/main/install.sh)"
  ```
