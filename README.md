# MINVIM

### Dependencies

##### Ensure you have `fzf` `curl` `nvim` and `rg`.
* Mac 
`brew install fzf curl nvim rg`

* Linux 
`apt-get install fzf curl nvim rg`

### Install
* Run
  ```
  mkdir -p ~/.config/nvim && \
    curl -o ~/.config/nvim/init.lua \
      https://raw.githubusercontent.com/jelohe/minvim/refs/heads/main/init.lua
  ```
